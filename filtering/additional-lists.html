Additional filter lists

This page shows page content that should be hidden or modified only when the user has enabled different filter lists In Brave this is done by enabling lists in brave://settings/shields/filters.

The below, right hand content should be hidden when the following are true.

  • Shields are enabled and "Block trackers and ads" is either default or aggressive.
  • The relevant filter list is enabled.
