Cookie Banners

This page shows page content that should be hidden or modified only when the user has enabled cookie notification blocking. In Brave this is done by enabling the cookie notification filter list in brave://settings/shields/filters, or accepting the "Cookie consent blocking" prompt.

The below, right hand content should be hidden when the following are true.

  • Shields are enabled and "Block trackers and ads" is either default or aggressive.
  • EasyList-Cookie List is enabled in brave://settings/shields/filters.
