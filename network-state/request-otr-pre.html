Request "off the record"

This tests whether Brave's Request "off the record" feature works as intended. The test works as follows:

  1. Step one: Set up

    Start this test in a clean profile.

  2. Step two: "Off the record" is enabled
    1. Click the link below to visit a "off the record" protected page on the other test domain, -.
    2. Choose to visit the page "off the record".
    3. Note that no stored values should appear in the table.
    4. Click the store test values button
    5. Confirm those new values appear recorded in the table.
    6. Click back to return to this page.
  3. Step three: Confirm "off the record" worked

    Check browsing history and confirm that no visits to - appear.

  4. Step four: "Off the record" is disabled
    1. Click the link below to visit a "off the record" protected page on the other test domain, -.
    2. Choose not to visit the page "off the record".
    3. Note that no stored values should appear in the table.
    4. Click the store test values button
    5. Confirm those new values appear recorded in the table.
    6. Click back to return to this page.
  5. Step five: Confirm normal behavior

    Check browsing history and confirm a visits to - now does appear.
